Pos Pricelist

Allows you to add price list in point of sale

Features
  • We can easily add price list in POS.
  • This module will work on offline mode.
1. Allow price list in POS and set default price list if you want.
2. List of price list.
3. Create new price list.
4. Create price list rule.
5. Select customer price list. When you change or set customer automatically selected price list will set.
6. Default selected price list.
7. List of price list in POS.
8. Customer price list in POS.
9. Selected price list in Pos order.
