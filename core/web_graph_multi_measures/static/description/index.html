Multi Measures

Currently, Odoo Graph view is not support show multiple measure on Bar & Line Chart. You only can select one measure at one time. So you cannot compare between 2 or more measures to analyse and appraise your work for improving.
Now, our module can help you present your data by show multi measures on graph view.
