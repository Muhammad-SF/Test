Website Multi Images

This module allows you to set multiple images for a product.
Select multiple images for a product in order to make it more attaractive.
Multiple images are displayed in a slider and you can choose an image from the slider.
Multiple images will displayed for Mobile also.
Multiple images will displayed for the product variants also.
Slider has buttons also to move back and forth.
